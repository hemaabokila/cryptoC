# Installation

We offer binaries ready to use for the following platforms (all are for 64 bits only):

* [Linux]
```
$ git clone https://github.com/hemaabokila/cryptoC.git
$ cd cryptoC
$ sudo mv cryptoC.py /usr/bin/cryptoC && chmod +x /usr/bin/cryptoC
$ cryptoC
```
